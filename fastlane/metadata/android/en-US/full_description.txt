<p><i>Thanox</i> is a system management tool that provides convenient features for system privacy and system optimization. It includes application permission management, background launch management, background run management, as well as powerful profile and component removal features.</p><p>First released from 2016, it was formerly known as <i>Application Management Xposed</i>, or <i>X-APM</i> for short, and later refactored to <i>Thanox</i> because the X-APM application architecture could not support continuous Android upgrades.</p><p><i>Thanox</i> aims to serve as a cell phone manager for native Android systems, focusing on application control, application privacy. Priority is given to supporting the latest three major versions of Android (currently up to Android 12L).</p><ul><li>It is not recommended to use Xiaobai</li><li>It is not recommended to use unstable Magisk and Xposed modules</li><li>Before updating you should take precautions for the case your device no longer starts.</li></ul>
<p><b>Note:</b> the app is not yet fully F/LOSS, see <a href='https://github.com/Tornaco/Thanox/issues/400' target='_blank' rel='nofollow'>this issue</a>. Hence the <code>NonFreeComp</code> anti-feature was applied.</p>
